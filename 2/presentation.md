# Основные функции структур данных

- `isempty()` — проверка на пустоту  
- `length()` — длина структуры  
- `in()` — проверка принадлежности  
- `unique()` — уникальные элементы  
- `reduce()` — свёртка структуры  
- `maximum()` / `minimum()` — поиск экстума
